Recipe Finder App 
React + Redux project using TheMealDB API

Description
-----------

A web application built with React and Redux that allows users to search, explore, and save cooking recipes using the TheMealDB API.

Features
--------
- Search recipes by ingredient
- View detailed recipe information
- Add and remove favorites
- Global state management with Redux
- Persistent favorites using Local Storage
- Responsive user interface
- Dynamic recipe rendering


 Technologies
 ------------

- React
- Redux
- React-Router
- JavaScript
- HTML5
- CSS3
- TheMealDB API
- Node.js
- Vite
